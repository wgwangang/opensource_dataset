# Open-Source Audio/Speech Datasets

A curated collection of open-source audio and speech datasets.

## File

- `audio_speech_datasets.csv` — Master CSV with all discovered datasets

## CSV Columns

| Column | Description |
|--------|-------------|
| dataset_name | Name of the dataset |
| duration_hours | Total audio/speech duration |
| language | Supported language(s) |
| description | Brief description of the dataset |
| license | License type |
| url | Official website or download link |

## Updates

This file is updated daily as new datasets are discovered.
